📖 우리반 칠판 사용 설명서

1. 수업 시작하기

  • 교사 화면에 로그인한 뒤, 상단 📱 참여 QR 코드를 눌러 학생들에게 QR을 보여 주세요.
  • 학생은 휴대폰으로 QR을 스캔해 /student 페이지에 들어갑니다.
  • 학생은 매번 이름과 자리 번호를 입력해야 참여할 수 있습니다.

2. 칠판에서 채팅 보기

  • 학생이 보낸 메시지는 해당 자리 아바타 아래 말풍선으로 표시됩니다.
  • 아바타를 클릭하면 마지막 채팅을 분필 목소리로 읽어 줍니다 🔊
  • 읽어 주면 학생의 🙋 손들기가 자동으로 내려갑니다. (다시 손들기 가능)

3. 학생 자리 버튼

  • 📷 — 프로필 사진 설정
  • ⚙️ — 채팅 기록 보기 · 수정 · 🔊 읽기
  • ✈️ — 이 학생에게만 DM 보내기
  • 🔇 — 음소거 (칠판에 안 보임) / 🔈 해제
  • 🚪 — 퇴장
  • 이름 옆 🙋 — 손들기 표시 · 클릭하면 내림

4. 도구 모음

  • 🧹 채팅 초기화 — 교사 칠판의 말풍선만 지웁니다 (학생 폰은 그대로)
  • ❓ 전체 질문 — 모든 학생 화면 중앙에 질문 표시
  • ⏱ 타이머 — 수업 시간 측정 · 종료 시 알람

5. 기타

  • 하단 👩‍🏫 선생님 입력란 — 칠판 20번(교사) 자리에 메시지 표시
  • 🪟 플로팅 창 — PPT 위에 작은 목록 창으로 보기
  • 🌙 — 칠판 / 밝은 칠판 테마 전환
  • 전체 강퇴 — 모든 학생을 한 번에 내보냅니다

6. 학생 화면 (카카오톡 스타일)
